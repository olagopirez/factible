# Código de conducta

Este proyecto adopta el [Contributor Covenant](https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/), versión 2.1.

## En resumen

- Tratá a todas las personas con respeto. Acá participan devs de todos los niveles, y las preguntas "básicas" son bienvenidas — integrarse con el Estado ya es suficientemente hostil.
- Criticá código e ideas, nunca personas.
- No se toleran acoso, ataques personales, lenguaje discriminatorio ni la publicación de información privada de terceros.
- Los desacuerdos técnicos se resuelven con argumentos y fuentes (las specs oficiales mandan), no a los gritos.

## Alcance

Aplica a todos los espacios del proyecto: issues, pull requests, discusiones y cualquier canal asociado, así como cuando se representa al proyecto en espacios públicos.

## Reportes

Si viviste o presenciaste una situación que viola este código, escribí a **olagopirez@gmail.com**. Los reportes se tratan con confidencialidad y se responden dentro de una semana.

Las consecuencias por violaciones siguen las [pautas de aplicación del Contributor Covenant](https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/#pautas-de-aplicaci%C3%B3n): de una corrección privada a la expulsión permanente del proyecto, según la gravedad.
