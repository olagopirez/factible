¿Preferís no gestionar certificados ni XML? Estamos evaluando un **gateway hosteado** sobre estas librerías:

- **API REST** — cualquier lenguaje integra por HTTP, sin SDK.
- **Certificados gestionados** — sin trámites de firma electrónica de tu lado.
- **Actualizaciones automáticas de formato** — cuando DGI cambia la spec, el gateway se actualiza.
- **Precio transparente por documento.**

Como las librerías son MIT y autohosteables, el gateway es una conveniencia, no un lock-in: siempre podés correr el mismo stack vos mismo.
